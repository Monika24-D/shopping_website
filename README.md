ONLINE SHOPPING WEBSITE

## Project Description
This project is a simple **online shopping website** developed using **HTML, CSS, and JavaScript combined in a single file (`index.html`)**.  
The website is designed with a **professional shopping layout** and uses a **slide-based navigation system**.

The website contains **10 slides**, and each slide opens one after another by clicking the **Next button**, creating a smooth and user-friendly shopping experience.  
The color theme used is **pink, blue, and white**, giving the website a modern and attractive appearance.

## Objectives
- To learn the basics of web development
- To design a professional shopping website interface
- To understand slide-based navigation using JavaScript
- To integrate HTML, CSS, and JavaScript in a single file
- To develop a beginner-friendly academic project

## Features
- Single-file implementation (`index.html`)
- 10 interactive slides
- Navigation using Next button
- Professional shopping website design
- Clean and attractive color theme
- Easy to use and understand

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
- The website opens with the first slide.
- Clicking the **Next** button displays the next slide.
- Slides are shown sequentially until the final slide.
- JavaScript controls slide navigation and interaction

## How to Run
1. Download the project files
2. Open `index.html` in any web browser
3. The website will run without any additional setup
